## Test Flow

1. **Sign Up**: Create a vendor account
2. **Settings**: Set store name, slug, and WhatsApp
3. **Products**: Add 2-3 products
4. **Store**: Visit `/s/your-slug` to see public store
5. **Order**: Place a test order as a customer
6. **Dashboard**: Check orders and analytics
